## 🎯 3-Minute Setup (For Experienced Users)

### Step 1: Create Shopify Store (5 min)
1. Sign up at [shopify.com](https://www.shopify.com)
2. Choose Basic plan (£24/month)
3. Connect your domain

### Step 2: Upload Theme (2 min)
```bash
# Create ZIP of theme files (exclude docs)
zip -r pdlab-theme.zip assets config layout locales sections snippets templates

# Upload via Shopify Admin:
# Online Store > Themes > Add theme > Upload zip file
```

### Step 3: Create Pages (10 min)
Go to **Online Store > Pages** and create 8 pages:

| Page Title | Template | URL Handle |
|------------|----------|------------|
| About Us | `page.about` | about |
| How It Works | `page.how-it-works` | how-it-works |
| Services | `page.services` | services |
| Pricing | `page.pricing` | pricing |
| Contact | `page.contact` | contact |
| Privacy Policy | `page.privacy-policy` | privacy-policy |
| Refund Policy | `page.refund-policy` | refund-policy |
| Shipping Policy | `page.shipping-policy` | shipping-policy |

### Step 4: Configure Theme (5 min)
Go to **Online Store > Themes > Customize**:
- Upload your logo
- Add contact information (phone, email, address)
- Customize colors (optional)
- Save changes

### Step 5: Set Up Navigation (3 min)
Go to **Online Store > Navigation**:

**Main Menu:**
- Home → /
- About Us → /pages/about
- How It Works → /pages/how-it-works
- Services → /pages/services
- Pricing → /pages/pricing
- Blog → /blogs/denture-care
- Contact → /pages/contact

**Footer Menu:**
- Privacy Policy → /pages/privacy-policy
- Refund Policy → /pages/refund-policy
- Shipping Policy → /pages/shipping-policy

### Step 6: Launch (1 min)
1. Go to **Online Store > Preferences**
2. Uncheck "Restrict access with password"
3. Save
4. Visit your website!

**Total time:** ~25 minutes

## 🆘 Need Help?

### Common Issues

**Problem:** Theme not appearing after upload
- **Solution:** Ensure ZIP only contains theme folders (assets, config, layout, locales, sections, snippets, templates)

**Problem:** Pages showing "Template not found"
- **Solution:** Select correct template from dropdown when creating pages

**Problem:** Images not loading
- **Solution:** Upload images via Settings > Files in Shopify

## 🎯 Quick Customization

### Change Colors
1. Go to **Customize Theme**
2. Find "Colors" section
3. Update Primary (blue) and Secondary (green) colors
4. Save

### Change Logo
1. Go to **Customize Theme**
2. Find "Logo" section
3. Upload your logo (200x50px recommended)
4. Adjust size if needed
5. Save

### Update Contact Info
1. Go to **Customize Theme**
2. Find "Contact Information" section
3. Update phone, email, address
4. Save

Changes appear immediately on your site!
